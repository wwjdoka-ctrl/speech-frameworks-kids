# Speech Frameworks for Kids

An interactive practice app for kids ages 4–12 to learn 9 speech and communication frameworks (I SAW, WHO+WHAT, STAR, PEP, PREP, SOAR, SBI, WHAT/SO WHAT/NOW WHAT, FAB), with age-appropriate examples across 8 everyday situations.

## How to use this on your phone

1. Open this site's GitHub Pages link in Safari (iPhone) or Chrome (Android).
2. **iPhone:** tap the Share icon → "Add to Home Screen".
3. **Android:** tap the ⋮ menu → "Add to Home screen" or "Install app".
4. It will appear as an app icon — tap to open, no browser bar.

No internet connection is needed once it's loaded — everything runs in the page itself.
